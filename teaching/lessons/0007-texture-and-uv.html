Texture + UV：把一張圖貼到模型上

Application Vertex Shader Rasterization Fragment Shader Output Merging
這堂課結束後，你會：
  • 知道 Texture = 貼在模型表面的 2D 圖，用 UV 對位
  • 知道採樣（sampling）就是「用 UV 去圖上取一個顏色」
  • 認得兩個常出 bug 的設定：Filtering（平滑）與 Wrapping（超界）

先看：圖怎麼貼上去

模型本身只有形狀。要有花紋、木紋、臉，就靠Texture（貼圖）—— 一張 2D 圖，貼到三角形表面上。這個技術叫 Texture Mapping。

UV：每個頂點記「我對應圖上哪一點」

第 2 課提過頂點屬性 UV：每個頂點帶一組 (u,v) ∈ [0,1]，指向貼圖上的一點。 光柵化把 UV 內插到每個片段，Fragment Shader 再用它去採樣（sampling）取顏色 —— 就是第 6 課看到的 texture2D(map, vTexCoord)。

兩個常出 bug 的設定

Filtering（過濾）—— 一個 UV 落在像素之間時，怎麼取色：

  • Nearest：取最近的一格 → 銳利、像素風，但放大會鋸齒
  • Linear：鄰近幾格平均 → 平滑，但可能糊

Wrapping（環繞）—— UV 超出 [0,1] 時怎麼辦：

  • Repeat：重複貼（磁磚、地板）
  • Clamp：夾在邊界（避免邊緣接縫）
貼圖鋸齒/糊多半是 Filtering；貼圖接縫/破圖/重複怪多半是 Wrapping 或 UV。 （另有 Mipmap：遠處自動換小張貼圖，減少閃爍 —— 先知道名字即可。）

快速確認

貼圖放大後一格一格有鋸齒，想要平滑，Filtering 該選？
