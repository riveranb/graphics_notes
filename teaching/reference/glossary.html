圖學渲染名詞對照表

這 9 課教到的名詞都收在這。之後的 lesson 一律沿用這裡的中英對照，不重新翻譯。

English中文一句話定義出處
Rendering渲染把 3D 模型（Mesh）變成一張 2D 圖像。L1
Graphics Pipeline繪圖管線3D→2D 之間一連串固定順序的處理階段。L1
Fixed-function固定管線GPU 照硬體既定方式處理、只能設定不能改演算法的階段。L1
Shader著色器管線中你可以寫程式（GLSL）控制的小程式。L1
Mesh網格由許多三角形拼成、逼近模型外形的資料。L1 / L2
Vertex頂點空間中的一個點，Mesh 的最小單位。L1 / L2
Triangle三角形3 個頂點圍成的面，Mesh 的基本組成。L1
Fragment片段光柵化後對應一個像素位置、帶顏色/深度等資訊的資料單位。L1 / L5
Vertex Attribute頂點屬性頂點攜帶的資料：Position、Normal、UV、Color 等。L2
Normal法向量垂直表面的方向向量，光照計算要用。L2 / L8
UV紋理座標頂點對應到貼圖上的座標 (u,v) ∈ [0,1]。L2 / L7
Index索引指定哪些頂點組成一個三角形，讓相鄰面共用頂點。L2
VBO / IBO / VAO頂點/索引/陣列緩衝物件把 Mesh 頂點與索引上傳、綁定給 GPU 的三種緩衝。L2
Winding Order繞序索引列出頂點的方向（CW 順時針 / CCW 逆時針），決定三角形哪面是正面。L2 / L5
Coordinate Space座標空間一組參照系：一個原點 + 一組軸，用來描述位置。L3
Object Space物件空間以模型自己為中心的座標（建模時）。L3
World Space世界空間以場景原點為中心，物件擺在世界哪裡。L3
Camera Space相機空間以相機為中心、從鏡頭看出去的座標。L3
Clip Space裁剪空間投影後的標準盒子 [-1,1]，超出的被裁掉。L3 / L4
Screen Space螢幕空間最終輸出裝置上的像素座標。L3
Perspective Division透視除法GPU 自動把座標除以 w，把 Clip Space 收斂成 NDC；發生在 clip 之後、rasterization 之前。L5 / L6
MVP Matrix模型/視圖/投影矩陣把頂點從 Object 一路換到 Clip Space 的矩陣（modelView × projection）。L3 / L4
Left / Right-handed左手/右手座標系X 右、Y 上時 Z 方向的兩種慣例：右手（OpenGL/Cocos）Z 朝外，左手（DirectX/Unity）Z 朝裡。L3
Perspective Projection透視投影近大遠小（模擬人眼），可視範圍是截頭金字塔。3D 場景用。L3
Orthographic Projection正交投影平行投影，遠近一樣大不變形，可視範圍是箱形。UI/2D/CAD 用。L3
View Frustum視錐體相機的可視範圍；透視投影下是一個截頭金字塔。L3 / L5
Vertex Shader頂點著色器每個頂點跑一次，算出頂點在 Clip Space 的位置。L1 / L4
attribute / uniform / varyingGLSL 變數修飾attribute=頂點輸入、uniform=全域輸入、varying=傳給下一階段的輸出。L4
gl_Position—Vertex Shader 必須輸出：頂點在 Clip Space 的位置。L4
Rasterization光柵化把三角形填成一格格片段，並內插頂點資料。L5
Interpolation內插由 3 個頂點的值，線性算出三角形內每個片段的值。L5
Frustum Culling視錐剔除相機視野外的整個物件在 CPU 端就丟掉。L5
Back Face Culling背面剔除背對相機的三角形面丟掉不畫。L5
Clipping裁剪跨出視野邊界的三角形，切掉超出的部分。L5
Fragment Shader片段著色器每個片段跑一次，算出最終像素顏色。L1 / L6
gl_FragColor—Fragment Shader 輸出：片段的最終顏色 (r,g,b,a)；預設值域夾在 [0,1]。L6
gl_FragCoord—Fragment Shader 唯讀內建輸入：片段在螢幕上的像素座標 (x,y)。L6
discard—Fragment Shader 指令：把當前片段整個丟掉（不寫顏色/深度），用於鏤空 / alpha test。L6
Early-Z提早深度測試GPU 在跑 fragment shader 前先做深度測試、剔除被擋住的片段；discard 或改深度會使它失效。L6 / L9
sampler2D貼圖採樣器Fragment Shader 用來讀取一張貼圖的 uniform 型別。L6 / L7
Texture貼圖貼到模型表面的 2D 圖像。L7
Texture Mapping貼圖映射用 UV 把 2D 貼圖對應到 3D 表面。L7
Sampling採樣用 UV 去貼圖上取一個顏色（Texel）。L7
Texel紋理像素貼圖的最小取樣單位，一格 (r,g,b,a)。UV × 解析度 = Texel 位置。L7
Anisotropic Filtering各向異性過濾沿被拉長方向多取樣，救回斜視角（地板/道路）遠處的清晰度；較貴。L7
Filtering過濾UV 落在像素之間如何取色：Nearest（銳利）/ Linear（平滑）。L7
Wrapping環繞模式UV 超出 [0,1] 如何處理：Repeat（重複）/ Clamp（夾邊）。L7
Mipmap多層級紋理預先做好的一組縮小貼圖，遠處自動換小張，減少閃爍。L7
Lighting光照用數學模型算每個表面點的明暗/顏色。L8
Ambient / Diffuse / Specular環境光 / 漫反射 / 高光Phong 光照的三個分量：基礎亮度 / 正對光較亮 / 光滑表面亮點。L8
PhongPhong 光照模型Ambient + Diffuse + Specular 三分量相加。L8
N·L (Lambert)法線·光向點積Diffuse 核心：表面法線與指向光源方向的點積＝有多正對光；背光 max(…,0) 夾成 0。L8
Shininess光澤度Specular 指數 n：越大高光越小越集中，表面看起來越光滑。L8
Blinn-PhongBlinn-Phong 模型Phong 改良版，高光改用半程向量算，更省更自然；引擎多用它。L8
Directional / Point / Spot平行光 / 點光 / 聚光三種基礎光源：太陽 / 燈泡（會衰減）/ 手電筒（錐形）。L8
Depth Test深度測試比較片段深度，近的才畫、遠的丟掉，自動解決遮擋。L9
Depth Buffer (Z-buffer)深度緩衝存每個像素「離鏡頭多遠」的緩衝區。L9
Blending顏色混合把新片段顏色和背景已有顏色混合，做透明/發光。L9
Alpha BlendingAlpha 混合來源色×α + 背景色×(1−α)，一般透明。L9
Additive加法混合直接把顏色相加，越疊越亮，火焰/粒子常用。L9
